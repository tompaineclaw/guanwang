# 建峰亿腾科技 — 官网

专注 OPC 一人公司项目孵化，助力独立创业者从 0 到 1。

## 本地开发

```bash
npm install
npm run dev          # http://localhost:5182
```

## 构建

```bash
npm run build        # 产物在 dist/
npm run preview      # 预览构建产物
```

## 技术栈

- Vite + React 18 + TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## 部署

通过 Netlify Git 集成自动部署。push 到 main 分支即触发。
